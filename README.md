# Parcial-Practico-Arquitectura-de-Software

Este repositorio presenta el desarrollo e implementación de una API RESTful para la gestión de notas estudiantiles. El sistema permite consultar y registrar calificaciones académicas mediante servicios web siguiendo los principios REST, versionamiento de API, validación de datos y documentación automática con OpenAPI/Swagger.
